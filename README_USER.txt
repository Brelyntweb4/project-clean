=== Brelynt Electron — Краткая инструкция пользователя ===

1. Скопируйте всю папку brelynt-electron на свой компьютер или флешку.
2. Установите Node.js (рекомендуемая версия — 18 и выше): https://nodejs.org/
3. Откройте папку brelynt-electron на вашем компьютере.
4. В этой папке запустите:
   - Для Windows: двойной клик по install-buy-module.ps1 (или через PowerShell: `powershell -ExecutionPolicy Bypass -File install-buy-module.ps1`)
   - Для Linux: терминал → `bash start-services.sh`
5. После запуска сервисов откройте файл `public/index.html` двойным кликом или через браузер.
6. Повторите эти шаги на каждом устройстве в вашей локальной сети.
7. Все устройства автоматически “увидят” друг друга — можно обмениваться токенами, файлами и сообщениями.
8. Приложение работает полностью офлайн, ваши данные не покидают ваш компьютер.
9. Для остановки сервиса просто закройте окно PowerShell/терминала или браузер.

Если возникнут вопросы — обратитесь к README.md или к поддержке на GitHub.

Удачи!
