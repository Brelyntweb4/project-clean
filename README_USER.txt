Web4 / Brelynt Electron — Тестовая лаборатория децентрализованного интернета

Что это такое?
---------------
Web4 — это автономная мини-сеть, которая объединяет ваши компьютеры и смартфоны в свой собственный интернет. 
Здесь вы сами управляете своими данными и ресурсами, нет центральных серверов, всё работает напрямую между вашими устройствами (P2P).

Возможности:
------------
- Используйте встроенный кошелёк для хранения и передачи токенов
- Обменивайтесь файлами с другими устройствами
- Получайте токены за предоставление памяти и вычислительных мощностей (гуманитарный майнинг)
- Вся сеть работает офлайн: без интернета, только внутри вашей локальной сети

Как запустить?
--------------
1. Скачайте архив приложения (web4.zip) и распакуйте на каждый компьютер и смартфон
2. Для компьютеров:
   - Запустите приложение через Electron или откройте файл `index.html` в браузере
3. Для смартфонов:
   - Подключитесь к той же Wi-Fi сети, что и компьютеры
   - В браузере введите адрес: `http://<IP_компьютера>:3040/mobile/`
4. Всё! Устройства найдут друг друга сами

Как проверить, что всё работает?
--------------------------------
- На каждом устройстве появится кошелёк и главное меню
- В списке участников должны быть видны остальные ваши устройства
- Можно попробовать отправить токены или файл между ними

Советы:
-------
- Все данные остаются на ваших устройствах, никакие облака не используются
- В любой момент можно добавить новые компьютеры или смартфоны — просто распакуйте архив и подключите к той же сети
- Если возникнут вопросы, посмотрите инструкцию на GitHub или напишите в раздел Issues: https://github.com/Brelyntweb4/web4/issues

Это лабораторная тестовая сеть Web4 (5 ПК + 3 смартфона) — ваша собственная цифровая экосистема!
